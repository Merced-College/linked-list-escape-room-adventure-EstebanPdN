# Escape Room Adventure (Linked List + GUI)

This project is an Escape Room Adventure Game built in Java.

## Features
- Custom Linked List to store scenes
- ArrayList for player inventory
- Data loaded from a CSV file
- Fully functional GUI using Java Swing
- Image display for each room
- Item pickup system
- Win condition based on collected items

## Updates Made
